# Sentinel Upload

Sentinel Upload is a secure file validation system implementing multi-layer verification including:

- Filename validation (Regex)
- Extension whitelist
- File size restriction
- Magic byte verification
- UUID-based secure renaming
- Logging of suspicious attempts

## Tech Stack
- Python
- Flask
- HTML/CSS

## How to Run
1. Install Flask
2. Run: python app.py
3. Open http://127.0.0.1:5000
